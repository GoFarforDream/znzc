# 智能助学生物产业专利案例课程整理报告

## 一、课程概况与基本信息

### 1.1 课程名称
**智能助学生物产业专利案例**

### 1.2 授课团队
- **主讲教师**：黄世英（所有互动课件）
- **项目负责人/研究主题**：葛福荣、蔡子荣
  - *注：两位负责人均指向同一核心课题——“新质生产力背景下生物产业专利转化与助学模式创新”。*

### 1.3 涉及学院与参与人数
- **覆盖学院**：港口与海岸工程学院、海洋食品与生物工程学院、海洋信息工程学院、海洋装备与机械工程学院、计算机工程学院、理学院、水产学院等。
- **当前加入人数**：60 人（加课码：BHG3GJ）。

### 1.4 课程定位
本课程旨在利用“智能助学”技术，结合生物产业特点，教授学生专利检索、分析、审查意见应对及案例应用技能，并推动新质生产力背景下的专利转化与助学模式创新。

| **第三章** | **资源检索方法** | 3.1 专利数据库检索<br>3.2 其他文献资源数据库检索<br>3.3 结合智能助学的检索策略<br>3.4 文献资源清单编制方法 |
| **第四章** | **审查意见书分析** | 4.1 发明人视角的审查注意事项<br>4.2 审查意见书及资料库检索<br>4.3 生物产业审查意见案例解析<br>4.4 智能助学辅助的策略设定 |

| **第八章** | **成果报告与总结** | 学生结合智能助学学习成果撰写课程报告书并进行总结。 |
