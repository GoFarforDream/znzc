# out 目录说明

本目录用于课程报告书的 LaTeX 写作与图表整理。

## 文件

- `写作思路与资料检索清单.md`：写作框架、资料检索建议、图表清单。
- `智能助产课程报告书模板.tex`：参照 Word 空白模板整理出的 LaTeX 大纲模板。
- `pic/`：后续存放所有图表文件。

## 最终提交命名

推荐最终文件名：

`第17组-课程报告书-葛福荣+蔡子荣.pdf`

若需提交 Word 源文件，则使用：

`第17组-课程报告书-葛福荣+蔡子荣.docx`

命名只保留“组别-文件用途-成员姓名”三层结构。软件编号 `2511` 已写入封面，不建议额外放入文件名，除非老师另有补充要求。

## 提交前合规核对

| 类别 | 风险点 | 修正要求 |
|---|---|---|
| 命名 | 使用“第几组”或范例姓名 | 改为 `第17组-课程报告书-葛福荣+蔡子荣` |
| 电子档 | 上传多个文件或压缩包 | 仅上传 1 个 PDF/Word 文件，图表嵌入正文 |
| 电子档 | 格式错乱 | 优先提交 PDF，同时保留 Word/LaTeX 源文件 |
| 纸本 | 单面打印或未装订 | 双面黑白打印，用订书针装订 |
| 纸本 | 一组提交多份 | 第17组只提交 1 份完整纸本 |
| 模板 | 删除原有黑色标题 | 保留模板标题，只在标题下补充正文，可增设小标题 |
| 字数 | 全文不足 5000 字 | 按写作框架填充，建议最终 6500-7500 字 |
| 参考文献 | 文献不足或不可检索 | 至少 6 篇中文真实文献，优先知网/万方/学校图书馆可检索 |
| 图表 | 空白或无来源 | 图表嵌入正文，非自制图表标注出处 |
| 口头报告 | 超时或偏离重点 | 2 分钟内，聚焦第5部分课程报告书总结 |

## 2分钟口头报告建议

课堂上台报告建议围绕第5部分展开，顺序可为：

1. 本组选题与研究对象。
2. 报告核心发现：智能助产如何体现生物产业专利转化和新质生产力。
3. 小组分工比例。
4. 图表数量与代表性图表。
5. AI 工具如何辅助检索、整理、写作和图表制作。
