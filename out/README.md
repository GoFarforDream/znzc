# out 目录说明

本目录用于课程报告书的 LaTeX 写作与图表整理。

## 文件

- `写作思路与资料检索清单.md`：写作框架、资料检索建议、图表清单。
- `智能助产课程报告书模板.tex`：参照 Word 空白模板整理出的 LaTeX 大纲模板。
- `pic/`：后续存放所有图表文件。

## 图表文件

LaTeX 模板中已预留以下文件名，请把对应图片放入 `pic/`：

- `fig01_technology_flow.png`
- `fig02_system_architecture.png`
- `fig03_clinical_scenarios.png`
- `tab01_patent_info.png`
- `tab02_literature_info.png`
- `tab03_comparison.png`
- `fig04_patent_layout.png`
- `tab04_problem_solution.png`
- `fig05_ai_workflow.png`

若暂时没有图表，可先注释 LaTeX 中对应的 `\includegraphics` 行，或先放入占位图。
